# Ace Agent 🎾

Live ATP tennis dashboard for the **Miami Open 2026**.

Features:
- Beautiful hard court dark theme (your original CSS)
- Today's Quarterfinals with simulated live odds drift
- Completed QF results (real from March 25-26, 2026)
- Tab navigation (Today / Player Search / News)
- One-click refresh

## Deploy on Streamlit (Free)

1. Click the **Deploy** button below or go to [share.streamlit.io](https://share.streamlit.io)
2. Connect this GitHub repo
